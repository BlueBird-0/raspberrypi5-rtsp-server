RTSP Media Server Documentation

소개

RTSP 프로토콜을 기반으로 한 미디어 스트리밍 서버 프로젝트입니다.

주요 기능

기능 목록

  • RTSP 프로토콜 기반 미디어 스트리밍
  • H.264 비디오 스트리밍 지원
  • Opus 오디오 스트리밍 지원
  • 멀티클라이언트 세션 관리
  • 실시간 미디어 프레임 캡처 및 버퍼링

아키텍처

주요 컴포넌트

  • RTSPServer: 전체 서버 관리
  • DataCapture: 미디어 프레임 캡처 및 버퍼 관리
  • MediaStreamHandler: 미디어 스트리밍 처리
  • ClientSession: 클라이언트 세션 관리

빌드 방법

* mkdir build
* cd build
* cmake ..
* make
