Detailed Description

RTP 헤더 정의 및 처리 클래스 헤더

RTP(Real-time Transport Protocol) 헤더의 구조체를 정의하고 처리하는 클래스

  • RTP 헤더 구조 정의 및 비트필드 처리
  • 네트워크 바이트 오더 변환
  • MTU 기반 패킷 크기 제한 처리
