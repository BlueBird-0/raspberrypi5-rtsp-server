\doxysubsection{Detailed Description}
H264 비디오 프레임 파싱 및 인코딩 클래스 헤더 

H264 포맷의 비디오 파일을 NAL(\+Network Abstraction Layer) Unit으로 파싱하고 프레임을 추출하는 기능을 제공하는 클래스
\begin{DoxyItemize}
\item mmap을 사용한 효율적인 파일 접근
\item H264 start code 인식 및 처리
\item NAL Unit 프레임 추출
\end{DoxyItemize}
